# Playtest Notes - v5

## 测试流程
1. ✅ 标题页 → 点"执笔出发" → 地图生成
2. ✅ 地图 → 点节点 → 进入战斗
3. ✅ 战斗 → 组句 → 吟诵 → 造成伤害/格挡
4. ✅ 战胜 → 奖励选牌 → 回到地图
5. ⏳ 休息/事件/商店/Boss 待测

## 句子评分实测
- 「李清照我」→ ×0.26（两主语无谓语=废句，吟诵禁用 ✅ 正确！）
- 「李清照斩」→ ×1.10 | ⚔️6（李清照诗意+0.3加成 ✅）
- 「李清照斩猛烈地」→ ×1.22 | ⚔️11（修饰词生效 ✅，语序有小扣分）
- 「我斩敌人」→ ×1.00 | ⚔️6（完整主谓宾 ✅）
- 「我斩敌人猛烈地」→ ×1.50 | ⚔️13（满配！✅）
- 「我守」→ ×0.85 | 🛡4（主谓缺宾 ✅）

## 已发现问题
1. ⚠️ 点击清除造句区的词不太直觉（需要点击造句区的词牌才能移除）
2. ⚠️ 句号选入造句区时有时排不到末尾（需要标点自动归位）
3. ⚠️ 语序不对时（如"我斩敌人猛烈地"修饰在宾语后面）给了×1.50，应该有扣分
