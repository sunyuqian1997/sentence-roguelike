# 《词灵录》水色OS 设计交付包 — 移植改造说明

> **目标**：照本包设计，把游戏现有界面全部替换成「水色OS」风格（celadon-and-persimmon retro OS theme）。
> **权威参考实现**（源码即规格，均可双击直接在浏览器打开，依赖同目录 `support.js` 与 `public/`）：
> - `战斗原型 3D 水色OS.dc.html` — 战斗主界面：全部布局、交互、动效、音效的唯一权威实现
> - `全套界面 水色OS.dc.html` — 标题 / 地图 / 词库 / 结算 四界面 + 组件与色彩规范（1e）+ 可交互反馈演示
> 两个文件的逻辑都在文件内 `class Component` 中，纯 JS + inline style + CSS keyframes，无构建依赖。模板内有逐块中文注释。

## 2. 窗口构件（所有界面所有面板统一）
1. **条纹标题栏**：`repeating-linear-gradient(0deg,#F6F1E2 0 2px,#1A1A18 2px 3px,#F6F1E2 3px 5px) content-box` + 白底居中标题「名.扩展名」（词灵戏台.pict / 造句.txt / 旅途.map / 词库.dict / 结算.log / menu.sys）。
2. **右上 ✕ 关闭键（可选组件）**：`margin-left:auto`，hover 反白、按下位移 1px。**常驻窗口不可关**——点击只播 392Hz 反馈音；仅弹窗/临时窗真正关闭。
3. **外圈衬边**：`box-shadow: 0 0 0 2px #C8B98A, <投影>`。
4. **投影=层级**：主窗 `7px 7px 0 rgba(20,42,38,.45)`，次窗 6px，小窗 5px，小件 3px。
5. **z 序**：窗口 3–6 ＜ 立绘 11（立绘永远压在窗口之上）＜ 文力/按钮面板 12 ＜ 大印 30 ＜ 飞行卡 40。

## 3. 动效与音效（规格 = 源码，搜 `tone(` / `thud(` / `@keyframes`）
### 窗口生命周期
- 开窗 `winIn`（居中窗用 `winInC`，keyframes 内含 translateX(-50%)）：`.3s cubic-bezier(.3,1.3,.4,1)` 回弹 + 660→920Hz 双音（间隔 90ms）
- 关窗 `winOut`：`.22s ease-in` 收拢 + 392Hz
- 界面进场：各窗 winIn **错峰 0 / .1 / .18 / .26s**（战斗界面顺序：戏台→造句→手牌→文力/按钮）
### 控件反馈（1e「反馈演示」卡有实机示意）
- hover：1180Hz tick + 抬起（卡牌 -14px / 图标 -3px）或反白
- 按下：`translate(2px,2px)` + 520Hz；禁用态 opacity .45
### 战斗演出时间轴（chantNow / endTurnNow）
- 吟诵逐字确认：每字 95ms，scale(1.45)+金光，五声音阶 notes=[392,440,494,523,587,659,698,784]
- 攻击句总长 ~1.5s：dim 聚光→台词气泡→前冲(260ms)→突刺命中(480ms，thud(180)+墨溅+shakeB+伤害飘字)→「中」印+「妙极」大印(860ms，thud(105)+880Hz)→复位(1350ms)
- 防御句 ~1.0s：蓝光吟诵(523Hz)→「守」印+盾/回血飘字(240ms，thud(200))→复位(780ms)
- 敌回合：意图闪烁+台词(330Hz square)→扑击(480ms)→命中(680ms，thud(140)，盾吸收结算)→复位(1200ms)
- 屏幕震动 shakeS/shakeB 必须放**内层包裹层**，绝不能与缩放 transform 同层
- WebAudio 需用户先交互一次（首次点击才建 AudioContext）

## 4. 言出法随（核心体验，必须实现）
- 词一**入句**（未吟诵）立即触发预备态，删字即退，切换 0.18–0.25s 缓动：
  - 句中出现攻击谓语 → 诗人 poetShift=110 前移 + charge 蓄力姿态（见 renderVals 的 `preview` 变量）
  - 攻击谓语+敌人本名同句 → 该敌人畏惧态（角色状态机详见另一交付包「小人表现力面板」）
